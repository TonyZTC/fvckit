======
FVCKIT
======

| **FVCKIT** is an open source package for Forensic Voice Comparison.
| It was forked from the SIDEKIT package (http://www-lium.univ-lemans.fr/sidekit/),
| an open source package for Speaker and Language Recognition and Diarization.
